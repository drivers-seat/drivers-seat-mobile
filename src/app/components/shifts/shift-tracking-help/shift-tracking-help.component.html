Tracking Your Mileage
Select "Allow While Using App" and turn on Precise location tracking.
Allow {{ appDisplayName }} to track your mileage in the background while you are using other apps.
You may be notified that Location Services are off. Press "Settings" to fix this.
Open location settings for {{ appDisplayName }}.
Set Allow Location Access to "Always" and turn-on Precise Location Tracking. Then, go back to settings.
When complete, it should look like this.
First allow "Precise" tracking and select "While using the app".
Next, "Allow" {{ appDisplayName }} to access your physical activity.
Now, allow {{ appDisplayName }} to track your mileage in the background while you are using other apps.
Confirm that location access is set to "Allow all the time". Hit back to finish.
When you start work, press this button to start tracking.
When tracking is turned on, the button will have a spinning outline.
And it will tell you how much work time and miles have been recorded.
When you're done, press the button again to turn off mileage tracking.
A summary of your mileage will appear.
